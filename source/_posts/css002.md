如何使用CSS做出：
1.左右布局
```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width">
  <title>JS Bin</title>
</head>
<body>
  <div class="container">
    <div class="left-half">
      左边
    </div>
    <div class="right-half">
      右边
    </div>
  </div>
</body>
</html>
```
使用背景渐变
```html
.container {
  background-image: 
    linear-gradient(
      to right,
      #f7941d 0%,
      #f7941d 50%,
      #8d5022 50%,
      #8d5022 100%);
}
```

使用绝对定位
```html
.container {
  width: 100%;
  position: relative;
}

.left-half {
  background-color: #f7941d;
  position: absolute;
  left: 0px;
  width: 50%;
}

.right-half {
  background-color: #8d5022;
  position: absolute;
  right: 0px;
  width: 50%;
}
```

使用假表
```html
.container {
  display: table;
}

.left-half,
.right-half {
  display:table-cell;
}

.left-half {
  background: #f7941d;
}

.right-half {
 background: #8d5022;
}
```

使用float
```html
.left-half,
.right-half {
  float: left;
}

.left-half {
  background: #f7941d;
  width: 50%;
}

.right-half {
  background: #8d5022;
  width: 50%;
}
注意，父元素需要class=clearfix,
因为float会使div脱离文档流，需要clearfix把后面正常参与文档流的内容后挪，
不然会造成div中的内容与后面的内容视觉上重合。
```

使用inline-block
```html
.left-half {
  background: #f7941d;
  display: inline-block;
}

.right-half {
  background: #8d5022;
  display: inline-block;
}
```

使用flexbox
```html
.container {
  display: flex;
  height: 100px;
}

.left-half,
.right-half {
  height: 100%;
}

.left-half {
  background: #f7941d;
  flex: 1;
}

.right-half {
  background: #8d5022;
  flex: 1;
}
```

使用grid
```html
.container {
  display: grid;
}

.left-half {
  background: #f7941d;
  grid-column: 1;
}

.right-half {
  background: #8d5022;
  grid-column: 2;
}
```

2.float + inline-block
```html
container {
    width:100%;
    text-align:center;
}

.left {
    float:left;
    width:200px;
    background: red;
}

.center {
    display: inline-block;
    margin:0 auto;
    width:200px;
    background: yellow;
}

.right {
    float:right;
    width:200px;
    background: blue;
}
```

3.flexbox css3
```html
.container {
  display: flex;
  flex-direction: row;
  flex-wrap: nowrap;
  justify-content: space-between;
  background-color: lightyellow;
}

.left ,
.center ,
.right {
  width: 100px;
  height: 100px;
  border: 2px dashed red;
}
```

水平居中
内联元素：
text-align: center;
块级元素：
margin: 0 auto;

垂直居中
1.不知道高度，绝对定位
```html
.parent {
    position:relative;
    }

.child {
    position: absolute;
    top: 50%;
    transform: translateY(-50%);
    }
```

2.父元素高度已知，相对定位
```html
.parent {
        height:xxx;
    }
    
.child {
    position: relative;
    top: 50%;
    transform: translateY(-50%);
    }
```

3.flex
```html
.parent {
    display:flex;/*Flex布局*/
    display: -webkit-flex; /* Safari */
    align-items:center;/*指定垂直居中*/
}
```
